# TIN-M12_ProjetoIntegrador
Projeto Integrador da Turma Técnico em Informática M12 do SENAC Gravataí

---
## 
Desenvolvimento de sistema web simples mas funcional para gestão de usuários, livros e reservas usando PHP8.

### Tecnologias
- Webserver: APACHE ou NGINX
- Banco de dados: MySQL, mas devido ao uso de PDO, podendo ser migrado facilmente para qualquer outro SGBD
- Linguagem de desenvolvimento: PHP8, sem qualquer outro framework com OOP e MVC.
